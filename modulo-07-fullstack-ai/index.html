🔧 Antes de empezar: herramientas necesarias

Instala estas herramientas antes de empezar la semana 9. Todo es gratuito.

Herramienta Para qué Acceso
Node.js 18+ Runtime para Next.js y el backend nodejs.org ↗
Next.js 14+ Framework React full-stack (npx create-next-app@latest) nextjs.org ↗
Vercel AI SDK Streaming de respuestas IA en React (npm install ai) ai-sdk.dev ↗
OpenAI API LLM para generar respuestas (Clases 01-05) platform.openai.com ↗
Auth.js (NextAuth) Autenticación OAuth y credenciales (npm install next-auth) authjs.dev ↗
Vercel Deploy gratuito de tu app vercel.com ↗
Instalación rápida: npx create-next-app@latest mi-ai-app && cd mi-ai-app && npm install ai @ai-sdk/openai next-auth
Coste estimado: 5-10 €/mes por uso de la API de OpenAI. Pon un límite de gasto mensual en tu dashboard de OpenAI.
