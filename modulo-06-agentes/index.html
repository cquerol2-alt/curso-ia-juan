Módulo 6: Agentes de IA

Semanas 8-9 · 12 horas · 6 clases

Objetivo del módulo

Los AI Agents son el futuro de la IA aplicada. Un agente es una IA que puede planificar, usar herramientas externas y ejecutar tareas de forma autónoma — no solo responde, sino que actúa.

De "responde a mi pregunta" a "investiga esto, analiza los datos y genera un informe": esta semana construirás agentes que trabajan solos y en equipo para resolver problemas complejos.

⚠️ Módulo en transición a v2: las clases 03 (CrewAI) y 04 (LangGraph) enseñan frameworks como alternativas. La filosofía canónica del curso en v2 es Anthropic SDK directo con tool use y agentic loops (ver clases 01 y 02, que son las columnas vertebrales). Trata los frameworks de las clases 03-04 como "conoce qué existe ahí fuera", no como el camino principal.

🔧 Antes de empezar: herramientas necesarias

Instala estas herramientas antes del lunes. Todo es gratuito y open source.

Herramienta Para qué Acceso
OpenAI API Function calling y tool use (Clases 01-05) platform.openai.com ↗
CrewAI Framework para agentes multi-agente (pip install crewai crewai-tools) docs.crewai.com ↗
LangGraph Agentes con estado y grafos (pip install langgraph langchain-openai) langchain.com/langgraph ↗
DuckDuckGo Search Herramienta de búsqueda web para agentes (pip install duckduckgo-search) PyPI ↗
Python-dotenv Gestión de API keys (pip install python-dotenv) PyPI ↗
Instalación rápida: pip install openai crewai crewai-tools langgraph langchain-openai duckduckgo-search python-dotenv
Coste estimado: 5-10 €/mes por uso de la API de OpenAI. Pon un límite de gasto mensual en tu dashboard de OpenAI.
