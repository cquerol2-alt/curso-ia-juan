📖 Resumen de conceptos

¿Qué es un Agente de IA?

Un agente de IA es un sistema que puede percibir su entorno, razonar y actuar de forma autónoma para alcanzar objetivos. A diferencia de un chatbot que solo responde preguntas, un agente puede:

  • Planificar: Dividir tareas complejas en pasos manejables
  • Usar herramientas: Ejecutar acciones (buscar web, hacer cálculos, consultar bases de datos)
  • Aprender: Mantener memoria de interacciones previas
  • Adaptarse: Ajustar su estrategia basado en resultados
📌 Chatbot vs Agente
Chatbot: "¿Cuál es la capital de Francia?" → Responde: "París"
Agente: "Necesito comparar precios de vuelos a París" → Busca precios, compara, sugiere opciones

1. LLM como Cerebro (Razonamiento)

El modelo de lenguaje es el "cerebro" del agente. Analiza la entrada, decide qué hacer y genera instrucciones para ejecutar acciones. En lugar de código hardcodeado, usa la inteligencia de un LLM (GPT-4, Claude, etc.) para razonar dinámicamente.

2. Tools (Herramientas / Function Calling)

Las herramientas son funciones que el agente puede invocar para afectar al mundo exterior. El LLM decide cuándo y cómo usarlas.

Ejemplo: herramientas disponibles 
tools = [

]

3. Planning (Planificación)

Los agentes buenos descomponen tareas complejas en pasos. Esto se llama chain-of-thought o reasoning.

Ejemplo de planificación:

Tarea: "Quiero ir a París. ¿Cuánto me cuesta un vuelo, dónde puedo dormir barato y qué monumentos debo visitar?"

Plan del agente:
  1. Buscar vuelos a París (search_web o API de vuelos)
  2. Buscar hoteles baratos en París (search_web)
  3. Buscar monumentos principales (search_web o base de datos)
  4. Compilar respuesta con toda la información

4. Memory (Memoria)

La memoria permite que el agente mantenga contexto a lo largo de múltiples interacciones.

Short-term Memory (Contexto): La conversación actual. Se mantiene en el prompt del LLM.

Long-term Memory (Persistencia): Información que persiste entre sesiones, almacenada en:
  • Vector stores (búsqueda semántica con embeddings)
  • Bases de datos estructuradas
  • Archivos o logs

El Ciclo Agentico Completo

Fase 1: PERCIBIR
Usuario: "¿Cuál es el precio de Bitcoin ahora y cuál era hace una semana?"

Fase 2: RAZONAR
LLM: "Necesito el precio actual (tool: get_bitcoin_price) y el precio histórico (tool: get_crypto_history)"

Fase 3: ACTUAR
Ejecuta: get_bitcoin_price() → $42,500
Ejecuta: get_crypto_history(date='1_week_ago') → $40,000

Fase 4: OBSERVAR
LLM recibe: {current: 42500, week_ago: 40000}

Fase 5: AJUSTAR
Respuesta: "Bitcoin subió un 6.25% en una semana, de $40,000 a $42,500"

Vamos a construir agentes simples en Python, empezando con pseudocódigo y evolucionando a código real.

Paso 1: El Loop Agentico Básico (Pseudocódigo)

Pseudocódigo 
FUNCIÓN agente_simple(pregunta_usuario):
    MIENTRAS no_completado:
        1. contexto = obtener_historial_conversación()

        2. respuesta_llm = enviar_al_llm(pregunta_usuario, contexto)

        3. SI respuesta_llm.requiere_tool:
            - herramienta = respuesta_llm.nombre_tool
            - parámetros = respuesta_llm.parámetros
            - resultado = ejecutar_tool(herramienta, parámetros)
            - contexto.añadir(f"Resultado tool: {resultado}")
            - pregunta_usuario = resultado  // siguiente iteración

        4. SINO:
            - DEVOLVER respuesta_llm
            - no_completado = FALSO

Paso 2: Agente Simple con OpenAI API

Este agente decide dinámicamente si necesita una herramienta o puede responder directamente.

Python 
import json
from openai import OpenAI

client = OpenAI(api_key="tu-api-key")

# Definir herramientas disponibles
tools = [
    {
        "type": "function",
        "function": {
            "name": "get_current_time",
            "description": "Obtiene la hora actual",
            "parameters": {
                "type": "object",
                "properties": {}
            }
        }
    },
    {
        "type": "function",
        "function": {
            "name": "calculator",
            "description": "Realiza operaciones matemáticas",
            "parameters": {
                "type": "object",
                "properties": {
                    "operation": {
                        "type": "string",
                        "description": "Operación a realizar (ej: 5 + 3)"
                    }
                },
                "required": ["operation"]
            }
        }
    }
]

# Funciones que el agente puede ejecutar
def get_current_time():
    from datetime import datetime
    return datetime.now().strftime("%H:%M:%S")

def calculator(operation):
    try:
        return str(eval(operation))
    except:
        return "Error en la operación"

# Loop del agente
def agent_loop(user_message):
    messages = [
        {"role": "user", "content": user_message}
    ]

    while True:
        # Llamar al LLM
        response = client.chat.completions.create(
            model="gpt-4",
            messages=messages,
            tools=tools,
            tool_choice="auto"
        )

        # Procesar respuesta
        if response.stop_reason == "tool_calls":
            # El LLM decidió usar una herramienta
            for tool_call in response.tool_calls:
                tool_name = tool_call.function.name
                tool_args = json.loads(tool_call.function.arguments)

                # Ejecutar la herramienta
                if tool_name == "get_current_time":
                    result = get_current_time()
                elif tool_name == "calculator":
                    result = calculator(tool_args["operation"])

                # Añadir resultado al contexto
                messages.append(
                    {"role": "assistant", "content": response.content}
                )
                messages.append(
                    {"role": "tool", "content": str(result),
                     "tool_call_id": tool_call.id}
                )
        else:
            # El LLM terminó y proporciona respuesta final
            final_response = response.choices[0].message.content
            print(f"Agente: {final_response}")
            break

# Usar el agente
agent_loop("¿Qué hora es y cuál es 25 * 4?")

Paso 3: Agente con Herramienta de Calculadora

Versión simplificada que muestra cómo un agente evalúa si una pregunta requiere una herramienta.

Python 
class SimpleAgent:
    def __init__(self):
        self.tools = {
            "calculator": self.calculator,
            "search": self.search,
            "translator": self.translator
        }

    def calculator(self, expression):
        """Herramienta: ejecuta cálculos matemáticos"""
        try:
            result = eval(expression)
            return f"Resultado: {result}"
        except:
            return "Error en la operación matemática"

    def search(self, query):
        """Herramienta: busca información (simulada)"""
        # En producción, llamarías a Google API o similar
        return f"Resultados para '{query}': [simulado]"

    def translator(self, text, language):
        """Herramienta: traduce texto (simulada)"""
        # En producción, usarías Google Translate API
        return f"Traducido a {language}: [simulado]"

    def decide_tool(self, user_input):
        """El agente decide qué herramienta usar"""
        if any(word in user_input.lower() for word in
               ["calcula", "operación", "matemática", "+"]):
            return "calculator"
        elif any(word in user_input.lower() for word in
                 ["busca", "información", "investiga"]):
            return "search"
        elif any(word in user_input.lower() for word in
                 ["traduce", "traducción"]):
            return "translator"
        return None

    def execute(self, user_input):
        """Loop principal del agente"""
        print(f"Usuario: {user_input}")

        # Paso 1: Decidir si necesita una herramienta
        tool_needed = self.decide_tool(user_input)

        if tool_needed:
            # Paso 2: Ejecutar la herramienta
            if tool_needed == "calculator":
                # Extraer la expresión de la pregunta
                expression = user_input.replace("Calcula", "").strip()
                result = self.calculator(expression)
            elif tool_needed == "search":
                result = self.search(user_input)
            elif tool_needed == "translator":
                result = self.translator(user_input, "inglés")

            # Paso 3: Generar respuesta final
            response = f"Usé la herramienta {tool_needed}. {result}"
        else:
            # Sin herramientas necesarias, responde directamente
            response = f"Entendido. La pregunta era: {user_input}"

        print(f"Agente: {response}\n")
        return response

# Usar el agente
agent = SimpleAgent()
agent.execute("Calcula 25 * 4")
agent.execute("Busca información sobre París")
agent.execute("Traduce 'Hello' al español")
