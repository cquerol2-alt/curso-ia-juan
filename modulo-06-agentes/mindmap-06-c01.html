Mind Map: Qué Son los Agentes de IA

← Volver a la Clase 01
Agente de IA ▼
Qué es: Sistema que percibe, razona y actúa de forma autónoma
Vs Chatbot: Chatbot responde, agente ejecuta acciones y resuelve problemas
Autonomía: Toma decisiones sin intervención humana constante
Iterativo: Aprende de resultados y ajusta estrategia
Componentes ▼
LLM: Modelo de lenguaje que razona y decide
Tools: Funciones externas que el agente puede usar
Planning: Estrategia para dividir tareas complejas
Memory: Almacenamiento de contexto e historial
Ciclo Agente ▼
1. Percibir: Recibe datos, contexto, entrada del usuario
2. Razonar: LLM analiza y decide qué hacer
3. Actuar: Ejecuta tool calls o genera respuesta
4. Observar: Recibe resultado de la acción
5. Ajustar: Itera si es necesario
Tool Use (Function Calling) ▼
Function Calling: LLM puede invocar funciones definidas
APIs: Acceso a servicios externos (búsqueda, base de datos)
Web Search: Buscar información en tiempo real
Structured Output: Respuestas en formato JSON controlado
Planning (Planificación) ▼
Descomposición: Dividir tarea grande en subtareas
Chain-of-Thought: Pensar paso a paso
ReAct: Reason (razonar) + Act (actuar) + Observe (observar)
Heurística: Usar estrategias probadas para problemas conocidos
Memory (Memoria) ▼
Short-term: Contexto de la conversación actual
Long-term: Vector store, base de datos persistente
RAG: Retrieval-Augmented Generation (busca + genera)
Embedding: Representación vectorial para búsqueda semántica
Tipos de Agentes ▼
Reactivos: Responden a estímulos sin memoria
Proactivos: Tienen objetivos y trabajan hacia ellos
Colaborativos: Múltiples agentes trabajando juntos
Híbridos: Combinan reactividad y proactividad
Casos de Uso ▼
Asistentes: Personal assistant, coding helper
Automation: RPA, procesos de negocio, ETL
Research: Investigar, recopilar, resumir información
Customer Support: Atención al cliente, ticketing, FAQ
