📚 Resumen de Conceptos Clave

¿Qué es CrewAI?

CrewAI es un framework de código abierto en Python para construir sistemas multi-agente donde varios agentes IA colaboran entre sí. A diferencia de un único agente, CrewAI permite orquestar equipos completos que trabajan juntos, delegando tareas y compartiendo información.

Conceptos Clave

Agent (Agente)

Entidad autónoma con:

  • Role: Su función en el equipo (ej: "Researcher", "Writer")
  • Goal: Su objetivo principal
  • Backstory: Contexto y personalidad que define cómo actúa
  • Tools: Herramientas que puede usar (búsqueda, web scraping, archivos, etc.)
  • Memory: Acceso a memoria corta y larga para aprender del contexto

Task (Tarea)

Trabajo específico asignado a un agente:

  • Description: Qué debe hacer
  • Expected Output: Qué esperas que devuelva (estructura y contenido)
  • Agent: Qué agente ejecuta la tarea
  • Tools: Herramientas específicas para esta tarea

Crew (Equipo)

Orquestador que coordina múltiples agentes y tareas:

  • Agents: Lista de agentes en el equipo
  • Tasks: Secuencia de tareas a ejecutar
  • Process: Cómo se ejecutan las tareas (Sequential o Hierarchical)
  • Verbose: Mostrar logs detallados durante la ejecución

Tools (Herramientas)

CrewAI proporciona herramientas integradas:

  • SerperDevTool: Buscar en Google (requiere API key gratis)
  • ScrapeWebsiteTool: Extraer contenido de páginas web
  • FileReadTool: Leer archivos locales
  • FileWriteTool: Crear/escribir archivos
  • DirectoryReadTool: Listar contenido de directorios
  • CodeInterpreterTool: Ejecutar código Python
  • Custom Tools: Puedes crear herramientas propias con @tool decorator

Procesos de Ejecución

Proceso Descripción Cuándo Usar
Sequential Las tareas se ejecutan una tras otra, en orden. Cada tarea ve los resultados de la anterior. Flujos lineales (ej: Investigar → Analizar → Escribir)
Hierarchical Un agente "Manager" coordina y delega tareas a otros agentes. Útil para parallelización y toma de decisiones complejas. Equipos grandes, problemas complejos que requieren coordinación

Memory en CrewAI

CrewAI soporta tres tipos de memoria:

  • Short-term Memory: Lo que pasó en la conversación actual
  • Long-term Memory: Información persistente entre conversaciones (requiere base de datos)
  • Entity Memory: Recuerda entidades específicas (personas, empresas, etc.)

Delegación

Un agente puede pedir ayuda a otros agentes del equipo. Esto permite descomponer problemas complejos de forma elegante.

¿CrewAI vs Agente Único?

Usa CrewAI cuando: Usa un Agente Único cuando:
Necesitas múltiples especializaciones (investigador, escritor, analista) La tarea es simple y centralizada
Las tareas son secuenciales o requieren coordinación Un agente puede hacer todo bien
Quieres explicabilidad (ver qué hace cada agente) Buscas máxima velocidad y simpleza
El workflow es complejo y se beneficia de separación de responsabilidades Los costos de tokens son críticos (CrewAI añade overhead)

🚀 Práctica Guiada: Tu Primer Crew

Objetivo: Crear un equipo de 2 agentes que investiga un tema y escribe un artículo.

Step 1: Instalación

Abre tu terminal y ejecuta:

pip install crewai crewai-tools

Opcionalmente, si quieres usar búsqueda web, también instala: pip install serper-dev

Step 2: Crear un Agente Simple (Researcher)

Crea un archivo crew_example.py:

from crewai import Agent, Task, Crew
from crewai_tools import SerperDevTool, ScrapeWebsiteTool

# Definir herramientas
search_tool = SerperDevTool()
scrape_tool = ScrapeWebsiteTool()

# Crear agente Researcher
researcher = Agent(
    role='Researcher',
    goal='Investigar temas en profundidad y recopilar información relevante',
    backstory='''Eres un investigador experiente con años de experiencia
    en búsqueda de información. Sabes cómo encontrar fuentes confiables
    y extraer los puntos clave de cualquier tema.''',
    tools=[search_tool, scrape_tool],
    verbose=True
)

print("✓ Agente Researcher creado")

Step 3: Crear un Crew con 2 Agentes

Añade a tu archivo (debajo del agente Researcher):

# Crear agente Writer
writer = Agent(
    role='Writer',
    goal='Escribir artículos claros, bien estructurados y atractivos',
    backstory='''Eres un escritor profesional especializado en crear
    contenido que enganche a la audiencia. Transformas información
    compleja en artículos claros y amenos.''',
    tools=[],
    verbose=True
)

# Crear tareas
research_task = Task(
    description='''Investiga sobre "Inteligencia Artificial Generativa en 2025".
    Busca:
    - Tendencias principales
    - Aplicaciones prácticas
    - Desafíos y oportunidades
    Recopila al menos 5 puntos clave.''',
    expected_output='''Un informe estructurado con al menos 5 puntos clave
    sobre IA Generativa en 2025, con fuentes.''',
    agent=researcher
)

writing_task = Task(
    description='''Basándote en la investigación del Researcher,
    escribe un artículo de 500-700 palabras sobre
    "Inteligencia Artificial Generativa en 2025".
    Estructura: Introducción, 3-4 secciones principales, Conclusión.''',
    expected_output='''Un artículo bien escrito, estructurado y listo
    para publicar en un blog.''',
    agent=writer
)

# Crear el Crew (Sequential)
crew = Crew(
    agents=[researcher, writer],
    tasks=[research_task, writing_task],
    process='sequential',
    verbose=True
)

# Ejecutar
result = crew.kickoff()
print("\n" + "="*60)
print("ARTÍCULO FINAL:")
print("="*60)
print(result)

Step 4: Ejecutar tu Primer Crew

En la terminal:

python crew_example.py

Nota: La ejecución tardará 30-60 segundos. Verás los logs de cada agente trabajando.

Step 5: Añadir Más Complejidad (Hierarchical)

Para un crew más avanzado con un Manager:

# Crear Manager
manager = Agent(
    role='Manager',
    goal='Coordinar el equipo para producir un artículo de excelente calidad',
    backstory='''Eres un manager experimentado que sabe cómo delegar
    y coordinar a especialistas. Tu objetivo es que el trabajo
    final supere las expectativas.''',
    verbose=True
)

# Crear el Crew (Hierarchical)
crew = Crew(
    agents=[manager, researcher, writer],
    tasks=[research_task, writing_task],
    process='hierarchical',
    manager_agent=manager,
    verbose=True
)

result = crew.kickoff()
⚠️ Importante: Para usar SerperDevTool, necesitas una API key gratuita de serper.dev. Guárdala como variable de entorno: export SERPER_API_KEY="tu_clave"
💡 Consejo: Empieza sin herramientas (tools=[]), luego añádelas gradualmente. Así ves cómo funciona la arquitectura básica antes de complejidad.

🎯 Mini-Reto: Crew de Análisis Tecnológico

Crea un crew de 3 agentes que trabaje juntos:

  1. Researcher: Investiga sobre un tema tech (ej: "Quantum Computing en 2025")
  2. Analyst: Analiza los hallazgos y extrae insights sobre tendencias e impacto
  3. Writer: Escribe un blog post de 800+ palabras con análisis y conclusiones

Requisitos:

  • Cada agente debe tener role, goal y backstory únicos
  • Las tareas deben encadenarse (sequential): Researcher → Analyst → Writer
  • El Writer debe recibir y refinar el trabajo del Analyst
  • Imprime el resultado final

Bonus: Crea un segundo crew con process='hierarchical' y añade un Manager. Compara los resultados.
