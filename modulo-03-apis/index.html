Objetivo del módulo

Conectar código propio con APIs de IA. Aprender a usar la API de OpenAI y la de Anthropic (Claude) desde Python. Construir un chatbot funcional con memoria de conversación y desplegarlo en la web.

De consumidor de IA a constructor: esta semana escribes código que habla con GPT y Claude.

🔧 Antes de empezar: cuentas y herramientas

Créate estas cuentas antes del lunes. Necesitarás acceso a las APIs de pago (con crédito gratuito de bienvenida).

Herramienta Para qué Alta
OpenAI Platform API key para llamar a GPT-4o desde Python (Clases 01-03, Boss Battle) platform.openai.com ↗
Anthropic Console API key para llamar a Claude desde Python (Clase 04, Boss Battle) console.anthropic.com ↗
Python 3.10+ Ya lo tienes del Módulo 1. Verificar con python --version python.org ↗
VS Code Editor de código. Extensión de Python instalada. code.visualstudio.com ↗
Render Deploy gratuito del chatbot (Boss Battle) render.com ↗
Coste estimado: OpenAI da 5$ de crédito gratis a cuentas nuevas. Anthropic también ofrece crédito gratuito. Con uso normal del curso, gastarás menos de 5€ en toda la semana. Pon un límite de gasto de 10€/mes en ambas plataformas para evitar sustos.
Variables de entorno: NUNCA pongas tu API key directamente en el código. Usa un archivo .env y la librería python-dotenv. Lo verás en la Clase 01.
