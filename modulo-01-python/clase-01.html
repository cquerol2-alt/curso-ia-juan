📖 Resumen

Variables y Asignación

En Python, las variables se crean simplemente asignando un valor. No necesitas declarar el tipo previamente (tipado dinámico).

Python 
nombre = "Juan"           # String (texto)
edad = 28                  # Integer (número entero)
altura = 1.75              # Float (número decimal)
es_desarrollador = True    # Boolean (verdadero/falso)

print(nombre)              # Juan
print(edad)                # 28
Comparación con JavaScript: En Python no usamos let, const o var. Simplemente asignas: nombre = "Juan". Python gestiona automáticamente la memoria.

Tipos de Datos Básicos

Python 
# Integer (int) — números enteros
edad = 25

en múltiples líneas"""

# Boolean (bool) — True o False
es_activo = True

Conversión de Tipos (Type Casting)

A veces necesitas convertir un tipo a otro. Python proporciona funciones para hacerlo:

Python 
# String a Integer
año_texto = "2026"
año_numero = int(año_texto)    # 2026 (ahora es número)

# Integer a String
edad = 28
edad_texto = str(edad)          # "28" (ahora es texto)

# String a Float
precio_texto = "19.99"
precio = float(precio_texto)    # 19.99 (número decimal)

# Cualquier tipo a Boolean
print(bool(1))                  # True
print(bool(0))                  # False
print(bool(""))                 # False (string vacío = False)
print(bool("hola"))             # True (string no vacío = True)

Operaciones con Strings

Python 
# Concatenación (juntar strings)
nombre = "Juan"
apellido = "Querol"
nombre_completo = nombre + " " + apellido
print(nombre_completo)          # Juan Querol

# Repetición
linea = "-" * 10
print(linea)                    # ----------

# Acceso a caracteres
palabra = "Python"
print(palabra[0])               # P (primer carácter)
print(palabra[2])               # t (tercer carácter)
print(palabra[-1])              # n (último carácter)

# Slicing (cortar partes)
codigo = "Python-3.11"
print(codigo[0:6])              # Python
print(codigo[7:])               # 3.11

# Métodos útiles
texto = "hola mundo"
print(texto.upper())            # HOLA MUNDO
print(texto.capitalize())       # Hola mundo
print(texto.replace("mundo", "Python"))  # hola Python
print(len(texto))               # 11 (longitud)

F-Strings (Formatted String Literals)

Los f-strings son la forma moderna y clara de insertar variables en strings. Son mucho más legibles que concatenación.

Python 
# Sintaxis: f"texto {variable}"
nombre = "Juan"
edad = 28
empresa = "Rice in Action"

# F-string simple
print(f"Hola, {nombre}")                    # Hola, Juan

# F-string con múltiples variables
print(f"{nombre} tiene {edad} años")        # Juan tiene 28 años

# F-string con expresiones
print(f"Año próximo tendré {edad + 1} años")  # Año próximo tendré 29 años

# F-string con formato numérico
precio = 19.999
print(f"Precio: ${precio:.2f}")             # Precio: $20.00

# F-string con padding (alineación)
print(f"Nombre: {nombre:20}")               # Nombre: Juan                 (alineado a la izquierda)
print(f"Edad: {edad:>3}")                   # Edad: 28 (alineado a la derecha)

# F-string con métodos
nombre = "juan"
print(f"Bienvenido, {nombre.upper()}")      # Bienvenido, JUAN
Pro tip: Siempre usa f-strings en lugar de concatenación con + o el método .format(). Son más rápidas y mucho más legibles.

Entrada de Usuario (input)

Para que tu programa sea interactivo, puedes pedir datos al usuario con input(). Siempre devuelve un string.

Python 
# input() siempre devuelve string
nombre = input("¿Cuál es tu nombre? ")
print(f"Hola, {nombre}")

# Si necesitas número, convierte
edad_texto = input("¿Cuántos años tienes? ")
edad = int(edad_texto)  # Conversión a número
edad_proximo = edad + 1
print(f"El próximo año tendrás {edad_proximo}")

# Versión compacta: conversión en la misma línea
año_nacimiento = int(input("¿En qué año naciste? "))
año_actual = 2026
edad_actual = año_actual - año_nacimiento
print(f"Naciste en {año_nacimiento}, así que tienes alrededor de {edad_actual} años")

Ejercicio 4: F-strings con formato

descuento = 0.10  # 10%

print(f"Total sin descuento: ${total_sin_descuento:.2f}")

Total a pagar: $54.00

Python 
¿Cuál es tu nombre? Cristina

Pistas:

  • Usa input() para leer el nombre (ya es string)
  • Usa input() + int() para leer el año (necesita conversión)
  • Calcula edad: 2026 - año_nacimiento
  • Usa f-string para imprimir el mensaje final
