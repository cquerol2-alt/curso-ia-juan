📖 Resumen: Flask Básico

¿Qué es Flask?

Flask es un micro-framework web ligero y flexible para Python. A diferencia de frameworks grandes como Django, Flask te deja total control sobre cómo estructurar tu aplicación. Es perfecto para APIs, servidores rápidos y proyectos medianos.

Flask vs Express.js

Ya conoces Express en Node.js. Flask funciona de forma muy similar: rutas, middlewares, respuestas JSON. La sintaxis cambia, pero la lógica es prácticamente igual. Flask es a Python lo que Express es a JavaScript.

Concepto Express.js Flask
Crear app const app = express() app = Flask(__name__)
Ruta GET app.get('/ruta', handler) @app.route('/ruta')
Respuesta JSON res.json({data}) jsonify({data})
Puerto app.listen(3000) app.run(port=5000)

Instalar Flask

Bash 
pip install flask

Tu Primera App: "Hola Mundo"

Python 
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return 'Hola Mundo'

if __name__ == '__main__':
    app.run(debug=True)

Explicación:

  • Flask(__name__) — Crea la app. __name__ indica dónde está el módulo.
  • @app.route('/') — Define una ruta (decorador).
  • debug=True — Recarga automáticamente en cambios y muestra errores detallados.

Rutas con Parámetros

Python 
@app.route('/usuario/<nombre>')
def saludar(nombre):
    return f'Hola, {nombre}'

# URL: /usuario/Juan → "Hola, Juan"

Métodos HTTP: GET y POST

Python 
@app.route('/api/mensaje', methods=['GET'])
def obtener_mensaje():
    return {'mensaje': 'GET funciona'}

@app.route('/api/crear', methods=['POST'])
def crear():
    return {'resultado': 'Dato creado'}, 201

Acceder a Datos (request object)

Python 
from flask import request

@app.route('/api/buscar', methods=['GET'])
def buscar():
    q = request.args.get('q')  # Query param
    return {'busca': q}

@app.route('/api/guardar', methods=['POST'])
def guardar():
    datos = request.json  # Body JSON
    return {'recibido': datos}

Devolver JSON con jsonify()

Python 
from flask import jsonify

@app.route('/api/usuario/<int:id>')
def usuario(id):
    return jsonify({'id': id, 'nombre': 'Juan', 'edad': 25})

Templates: HTML Dinámico

Usa render_template() para renderizar HTML con variables de Python.

Python 
from flask import render_template

@app.route('/perfil/<nombre>')
def perfil(nombre):
    return render_template('perfil.html', nombre=nombre, edad=25)

En templates/perfil.html:

HTML (Jinja2) 
<h1>Perfil de {{ nombre }}</h1>
<p>Edad: {{ edad }}</p>

Debug Mode

debug=True activa:

  • Recarga automática (hot reload).
  • Debugger interactivo en errores.
  • Solo en desarrollo. En producción, debug=False.
⚠️ Importante: Nunca uses debug=True en producción. Expone información sensible del servidor.

💻 Práctica Guiada: App Flask Paso a Paso

Paso 1: Hola Mundo

Crea un archivo app.py:

Python 
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return 'Hola Mundo'

if __name__ == '__main__':
    app.run(debug=True)

Ejecuta: python app.py

Abre en navegador: http://localhost:5000

Paso 2: Ruta con Parámetros

Python 
@app.route('/saludo/<nombre>')
def saludo(nombre):
    return f'¡Hola, {nombre}! Bienvenido a Flask.'

Prueba: http://localhost:5000/saludo/Juan

Paso 3: Endpoint JSON

Python 
from flask import jsonify

@app.route('/api/usuario/<nombre>')
def api_usuario(nombre):
    return jsonify({
        'nombre': nombre,
        'lenguaje': 'Python',
        'framework': 'Flask'
    })

Prueba: http://localhost:5000/api/usuario/Juan → JSON

Paso 4: Manejo de Formularios (POST)

Python 
from flask import request

@app.route('/api/registrar', methods=['POST'])
def registrar():
    datos = request.json
    nombre = datos.get('nombre')
    email = datos.get('email')

    return jsonify({
        'mensaje': f'Registrado: {nombre}',
        'email': email,
        'status': 'ok'
    }), 201

Prueba con curl:

Bash 
curl -X POST http://localhost:5000/api/registrar \
  -H "Content-Type: application/json" \
  -d '{"nombre": "Juan", "email": "juan@example.com"}'

🧪 Mini-reto: 3 Endpoints Flask

Crea una app Flask con estos 3 endpoints:

1. GET / (Inicio)

Devuelve un HTML o texto simple: "Bienvenido a mi servidor Flask"

2. GET /api/saludo/<nombre>

Devuelve JSON:

JSON 
{
    "mensaje": "Hola, [nombre]",
    "saludo_completo": "¡Hola, [nombre]! Bienvenido a mi API Flask.",
    "timestamp": "2026-04-03T10:30:00"
}

3. GET /api/hora

Devuelve la hora actual en JSON:

JSON 
{
    "hora": "2026-04-03T10:30:45",
    "timezone": "UTC",
    "formato": "ISO 8601"
}

Pista: Usa from datetime import datetime

Solución completa:
Python 
from flask import Flask, jsonify
from datetime import datetime

app = Flask(__name__)

@app.route('/')
def home():
    return 'Bienvenido a mi servidor Flask'

@app.route('/api/saludo/<nombre>')
def saludo(nombre):
    return jsonify({
        'mensaje': f'Hola, {nombre}',
        'saludo_completo': f'¡Hola, {nombre}! Bienvenido a mi API Flask.',
        'timestamp': datetime.now().isoformat()
    })

@app.route('/api/hora')
def hora():
    return jsonify({
        'hora': datetime.now().isoformat(),
        'timezone': 'UTC',
        'formato': 'ISO 8601'
    })

if __name__ == '__main__':
    app.run(debug=True)
