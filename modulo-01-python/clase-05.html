📖 Resumen

Manejo de ficheros

  • open() — Abre un archivo. Modos: 'r' (lectura), 'w' (escritura), 'a' (añadir), 'b' (binario)
  • read(), readline(), readlines() — Lectura de contenido
  • write(), writelines() — Escritura de contenido
  • with statement — Abre y cierra ficheros automáticamente (recomendado)
with open('datos.txt', 'r') as archivo:
    contenido = archivo.read()

with open('salida.txt', 'w') as archivo:
    archivo.write('Hola, fichero!')

Módulo JSON

  • json.loads() — Convierte string JSON a objeto Python
  • json.dumps() — Convierte objeto Python a string JSON
  • json.load() — Lee JSON desde un fichero
  • json.dump() — Escribe JSON en un fichero
import json

# De string a dict
datos_json = '{"nombre": "Juan", "edad": 25}'
datos = json.loads(datos_json)

# De dict a string
usuario = {"nombre": "María", "edad": 30}
json_string = json.dumps(usuario)

# Desde fichero
with open('usuarios.json', 'r') as f:
    usuarios = json.load(f)

# A fichero
with open('usuarios.json', 'w') as f:
    json.dump(usuarios, f, indent=2)

Librería requests (APIs)

  • requests.get() — Realiza petición GET
  • requests.post() — Realiza petición POST
  • headers — Encabezados HTTP personalizados
  • response.json() — Parsea respuesta JSON automáticamente
  • response.status_code — Código de estado HTTP
import requests

# GET simple
response = requests.get('https://api.example.com/users')
datos = response.json()

# GET con parámetros
params = {'page': 1, 'limit': 10}
response = requests.get('https://api.example.com/users', params=params)

# POST con JSON
payload = {'nombre': 'Juan', 'email': 'juan@example.com'}
headers = {'Content-Type': 'application/json'}
response = requests.post('https://api.example.com/users',
                         json=payload,
                         headers=headers)

# Verificar resultado
if response.status_code == 200:
    datos = response.json()
else:
    print(f'Error {response.status_code}')

Comparación: fetch() (JavaScript) vs requests (Python)

Aspecto JavaScript (fetch) Python (requests)
Sintaxis GET fetch(url) requests.get(url)
Parsear JSON response.json() (async) response.json() (síncrono)
Headers headers: {...} en options headers={...} parámetro
POST method: 'POST', body: JSON.stringify(...) requests.post(..., json=...)

import json

# Crear diccionario y guardarlo en JSON
estudiante = {
    'nombre': 'Juan',
    'edad': 25,
    'cursos': ['Python', 'JavaScript', 'SQL'],
    'calificacion': 8.5
}

# Guardar a fichero
with open('estudiante.json', 'w', encoding='utf-8') as f:
    json.dump(estudiante, f, indent=2, ensure_ascii=False)

# Cargar desde fichero
with open('estudiante.json', 'r', encoding='utf-8') as f:
    datos_cargados = json.load(f)
    print(datos_cargados)
    print(f"Nombre: {datos_cargados['nombre']}")

🧪 Mini-reto

Objetivo: Crear un script que llame a una API pública, guarde los datos en JSON y luego los lea.

Requisitos:

  1. Llamar a la API de Random User Generator
  2. Obtener 5 usuarios aleatorios
  3. Guardar los datos en usuarios.json
  4. Leer el fichero y mostrar nombre y email de cada usuario
  5. Añadir manejo de errores básico

Solución sugerida:

import requests
import json

def descargar_usuarios(cantidad=5):
    """Descarga usuarios de la API Random User"""
    try:
        url = f'https://randomuser.me/api/?results={cantidad}'
        response = requests.get(url, timeout=5)
        response.raise_for_status()
        return response.json()
    except requests.exceptions.RequestException as e:
        print(f"Error en la petición: {e}")
        return None

def guardar_en_json(datos, nombre_archivo='usuarios.json'):
    """Guarda los datos en un fichero JSON"""
    try:
        with open(nombre_archivo, 'w', encoding='utf-8') as f:
            json.dump(datos, f, indent=2, ensure_ascii=False)
        print(f"✓ Datos guardados en {nombre_archivo}")
    except IOError as e:
        print(f"Error escribiendo fichero: {e}")

def leer_y_mostrar(nombre_archivo='usuarios.json'):
    """Lee el fichero y muestra nombre y email"""
    try:
        with open(nombre_archivo, 'r', encoding='utf-8') as f:
            datos = json.load(f)

        print("\n📋 Usuarios cargados:\n")
        for i, usuario in enumerate(datos['results'], 1):
            nombre = f"{usuario['name']['first']} {usuario['name']['last']}"
            email = usuario['email']
            print(f"{i}. {nombre} — {email}")
    except FileNotFoundError:
        print(f"Fichero {nombre_archivo} no encontrado")
    except json.JSONDecodeError:
        print("Error leyendo JSON")

# Ejecutar
if __name__ == "__main__":
    print("🔄 Descargando usuarios...")
    datos = descargar_usuarios(5)

    if datos:
        guardar_en_json(datos)
        leer_y_mostrar()
