📖 Resumen

¿Qué es Web Scraping?

Web scraping es la técnica de extraer datos automáticamente de páginas web. En lugar de copiar-pegar manualmente, usamos código para obtener información estructurada de HTML.

Caso de uso: Extraer precios de productos, noticias, datos públicos, reviews, cotizaciones bursátiles, etc.

El combo: Requests + BeautifulSoup

Para hacer web scraping en Python necesitamos dos librerías:

  • requests: Descarga el contenido HTML de una URL
  • BeautifulSoup: Analiza y navega el HTML para extraer datos
Python 
# Instalar librerías
pip install requests beautifulsoup4

# Importar
import requests
from bs4 import BeautifulSoup

Flujo básico de scraping

Python 
# 1. Descargar la página
response = requests.get('https://ejemplo.com')
html_content = response.text

# 2. Parsear el HTML
soup = BeautifulSoup(html_content, 'html.parser')

# 3. Extraer datos
titulo = soup.find('h1').text

Parsing HTML: find() vs find_all()

BeautifulSoup te permite buscar elementos en el HTML:

Python 
soup = BeautifulSoup(html, 'html.parser')

# find() → obtiene el PRIMER elemento que coincide
primer_parrafo = soup.find('p')

# find_all() → obtiene TODOS los elementos que coinciden
todos_parrafos = soup.find_all('p')

# Buscar por atributos
enlace_principal = soup.find('a', {'class': 'main-link'})

# Buscar anidado
contenedor = soup.find('div', {'id': 'content'})
articulos = contenedor.find_all('article')

CSS Selectors (aprovecha tu conocimiento CSS)

Si ya conoces CSS, puedes usar select() con los mismos selectores:

Python 
# Por clase
soup.select('.producto')  # Todos con clase 'producto'
soup.select_one('.producto')  # El primero

# Por ID
soup.select('#header')

# Selectores combinados
soup.select('div.card > p.titulo')  # Párrafos dentro de divs con clase card
soup.select('a[href*="github"]')  # Enlaces cuyo href contiene 'github'

Navegando el árbol HTML

Python 
elemento = soup.find('h2')

# Acceder a parent (elemento contenedor)
contenedor = elemento.parent

# Acceder a hermanos
siguiente = elemento.next_sibling
anterior = elemento.previous_sibling

# Acceder a hijos
primer_hijo = elemento.contents[0]

Extrayendo texto y atributos

Python 
enlace = soup.find('a')

# Obtener texto limpio
texto = enlace.text  # o .get_text()

# Obtener atributos
href = enlace['href']  # Acceso directo
clase = enlace.get('class')  # Seguro (devuelve None si no existe)
todos_attrs = enlace.attrs  # Diccionario de todos

# HTML limpio
print(enlace.string)  # Solo si tiene UN hijo de texto

💻 Práctica guiada: Scraper de Quotes

Vamos a extraer citas del sitio público https://quotes.toscrape.com y guardarlas en un archivo JSON.

Objetivo

Scrapeador que extrae: autor, texto de la cita y tags. Guardamos en JSON.

Código paso a paso

Python 
import requests
from bs4 import BeautifulSoup
import json

# 1. Descargar la página
url = 'https://quotes.toscrape.com'
response = requests.get(url)
soup = BeautifulSoup(response.content, 'html.parser')

# 2. Encontrar todas las citas
quotes_data = []

# Cada cita está en un div con clase 'quote'
for quote_div in soup.find_all('div', {'class': 'quote'}):

    # Extraer el texto (está en span con clase 'text')
    text = quote_div.find('span', {'class': 'text'}).text

    # Extraer el autor (está en small con clase 'author')
    author = quote_div.find('small', {'class': 'author'}).text

    # Extraer tags (cada uno en un 'a' dentro de 'div.tags')
    tags = []
    for tag_a in quote_div.find_all('a', {'class': 'tag'}):
        tags.append(tag_a.text)

    # Guardar en estructura
    quotes_data.append({
        'text': text,
        'author': author,
        'tags': tags
    })

# 3. Guardar en JSON
with open('quotes.json', 'w', encoding='utf-8') as f:
    json.dump(quotes_data, f, ensure_ascii=False, indent=2)

print(f'✅ Se extrajeron {len(quotes_data)} citas')
