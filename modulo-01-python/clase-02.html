Resumen

Listas (Lists)

Las listas son colecciones ordenadas de elementos, mutables (se pueden modificar). Se crean con corchetes [].

# Crear una lista
numeros = [1, 2, 3, 4, 5]
nombres = ["Alice", "Bob", "Charlie"]
mixta = [1, "dos", 3.0, True]

# Acceder por índice (empieza en 0)
print(numeros[0])      # 1
print(nombres[1])      # "Bob"
print(numeros[-1])     # 5 (último elemento)

Slicing (Cortes)

Extrae un rango de elementos usando [inicio:fin:paso].

numeros = [0, 1, 2, 3, 4, 5]
print(numeros[1:4])      # [1, 2, 3] (hasta 4 no incluido)
print(numeros[:3])       # [0, 1, 2] (desde el inicio)
print(numeros[2:])       # [2, 3, 4, 5] (hasta el final)
print(numeros[::2])      # [0, 2, 4] (cada 2 elementos)
print(numeros[::-1])     # [5, 4, 3, 2, 1, 0] (invertida)

Métodos de Listas

numeros = [1, 2, 3]

numeros.append(4)           # [1, 2, 3, 4] - añade al final
numeros.insert(0, 0)        # [0, 1, 2, 3, 4] - inserta en posición
numeros.pop()               # saca y retorna el último
numeros.remove(2)           # elimina el primer 2 que encuentra
numeros.sort()              # ordena la lista (modifica original)
numeros.reverse()           # invierte la lista
numeros.clear()             # vacía la lista
numeros.copy()              # crea una copia
numeros.index(1)            # índice del primer 1 (0)
numeros.count(1)            # cuántas veces aparece 1

Tuplas (Tuples)

Las tuplas son colecciones ordenadas, inmutables (no se pueden modificar). Se crean con paréntesis () o comas.

# Crear tuplas
coordenadas = (10, 20)
colores = ("rojo", "verde", "azul")
una_sola = (42,)             # ¡coma necesaria para una sola!

# Acceso igual que listas
print(coordenadas[0])         # 10
print(colores[-1])            # "azul"

# Slicing igual que listas
print(coordenadas[0:1])       # (10,)

# NO puedes modificar
# coordenadas[0] = 15         # ❌ TypeError!

Desempaquetado (Unpacking)

Extrae elementos de una tupla a múltiples variables.

# Desempaquetado simple
x, y = (10, 20)
print(x, y)                   # 10 20

a, b, c = (1, 2, 3)
print(a, b, c)                # 1 2 3

# Con asterisco (coge el resto)
first, *rest, last = (1, 2, 3, 4, 5)
print(first)                  # 1
print(rest)                   # [2, 3, 4]
print(last)                   # 5

Diccionarios (Dictionaries)

Los diccionarios almacenan pares clave-valor. Se crean con llaves {}. Las claves son únicas e inmutables; los valores pueden ser cualquier tipo.

# Crear diccionarios
persona = {
    "nombre": "Alice",
    "edad": 28,
    "ciudad": "Madrid",
    "activo": True
}

# Acceso por clave
print(persona["nombre"])      # "Alice"
print(persona.get("edad"))    # 28
print(persona.get("email", "N/A"))  # "N/A" (no existe)

# Modificar
persona["edad"] = 29          # actualizar
persona["email"] = "alice@example.com"  # agregar

# Eliminar
del persona["ciudad"]
persona.pop("activo")         # saca y retorna el valor

Métodos de Diccionarios

persona = {"nombre": "Bob", "edad": 30}

persona.keys()                # dict_keys(['nombre', 'edad'])
persona.values()              # dict_values(['Bob', 30])
persona.items()               # dict_items([...])

persona.update({"email": "bob@ex.com"})
persona.clear()               # vacía el diccionario
persona.copy()                # crea una copia

# Comprobar existencia
if "nombre" in persona:       # True
    print(persona["nombre"])

Diccionarios Anidados

empresa = {
    "nombre": "Tech Corp",
    "empleados": [
        {"id": 1, "nombre": "Alice", "rol": "Dev"},
        {"id": 2, "nombre": "Bob", "rol": "PM"}
    ],
    "ubicaciones": {
        "madrid": "Calle Mayor 10",
        "barcelona": "Paseo de Gracia 50"
    }
}

print(empresa["empleados"][0]["nombre"])  # "Alice"
print(empresa["ubicaciones"]["madrid"])   # "Calle Mayor 10"

Comparación: Python vs JavaScript

Concepto Python JavaScript
Array / Lista list = [1, 2, 3] const list = [1, 2, 3]
Tupla (inmutable) tuple = (1, 2, 3) const tuple = [1, 2, 3] (con const)
Objeto / Diccionario dict = {"key": "value"} const obj = {key: "value"}
Acceso list[0], dict["key"] list[0], obj.key o obj["key"]
Métodos .append(), .pop(), .keys() .push(), .pop(), Object.keys()

Ejercicio 1: Manipula Listas

# Crea una lista de números
numeros = [10, 20, 30, 40, 50]

# Añade 60 al final
numeros.append(60)

# Inserta 5 al principio (índice 0)
numeros.insert(0, 5)

# Saca el último elemento
ultimo = numeros.pop()

# Ordena la lista
numeros.sort()

# Invierte la lista
numeros.reverse()

print(numeros)  # ¿Qué sale?
Ver solución 
# Solución
numeros = [10, 20, 30, 40, 50]
numeros.append(60)
numeros.insert(0, 5)
ultimo = numeros.pop()
numeros.sort()
numeros.reverse()
print(numeros)  # [60, 50, 40, 30, 20, 10, 5]

Ejercicio 2: Tuplas y Desempaquetado

# Crea una tupla de coordenadas
punto = (100, 200)

# Desempaqueta en x e y
x, y = punto

# Crea otra tupla con más elementos
valores = (1, 2, 3, 4, 5)

# Desempaqueta: primero, resto y último
primero, *resto, ultimo = valores

print(f"x={x}, y={y}")
print(f"primero={primero}, resto={resto}, ultimo={ultimo}")
Ver solución 
# Solución
punto = (100, 200)
x, y = punto
valores = (1, 2, 3, 4, 5)
primero, *resto, ultimo = valores
print(f"x={x}, y={y}")  # x=100, y=200
print(f"primero={primero}, resto={resto}, ultimo={ultimo}")
# primero=1, resto=[2, 3, 4], ultimo=5

Ejercicio 3: Diccionarios

# Crea un diccionario de un producto
producto = {
    "nombre": "Laptop",
    "precio": 1200,
    "stock": 5,
    "disponible": True
}

# Accede al precio
print(producto["precio"])

# Actualiza el stock (restale 1)
producto["stock"] -= 1

# Añade una clave nueva "descuento"
producto["descuento"] = 0.1

# Imprime todas las claves
print(producto.keys())

# Imprime todos los valores
print(producto.values())
Ver solución 
# Solución
producto = {
    "nombre": "Laptop",
    "precio": 1200,
    "stock": 5,
    "disponible": True
}
print(producto["precio"])  # 1200
producto["stock"] -= 1
producto["descuento"] = 0.1
print(producto.keys())  # dict_keys(['nombre', 'precio', 'stock', ...])
print(producto.values())  # dict_values(['Laptop', 1200, 4, ...])

Mini-Reto: Libreta de Contactos

Crea una pequeña libreta de contactos usando un diccionario. Cada contacto debe tener nombre, teléfono y email.

Requisitos:

  1. Crea un diccionario contactos donde la clave es el nombre y el valor es otro diccionario con teléfono y email.
  2. Añade al menos 3 contactos.
  3. Imprime el contacto de una persona (acceso anidado).
  4. Modifica el teléfono de uno de los contactos.
  5. Itera sobre todos los contactos e imprime en formato legible.
Ver solución propuesta 
contactos = {
    "Alice": {
        "telefono": "123-456-7890",
        "email": "alice@example.com"
    },
    "Bob": {
        "telefono": "098-765-4321",
        "email": "bob@example.com"
    },
    "Charlie": {
        "telefono": "555-555-5555",
        "email": "charlie@example.com"
    }
}

# Acceso anidado
print(f"Teléfono de Alice: {contactos['Alice']['telefono']}")

# Modificar
contactos["Bob"]["telefono"] = "111-111-1111"

# Iterar
for nombre, datos in contactos.items():
    print(f"{nombre}: {datos['email']} | {datos['telefono']}")
