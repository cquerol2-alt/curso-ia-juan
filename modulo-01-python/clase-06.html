CS50P — Sección de Object-Oriented Programming

📖 Resumen

¿Qué es la Programación Orientada a Objetos?

La OOP es un paradigma que organiza el código en objetos que contienen datos (atributos) y comportamientos (métodos). En Python, los objetos se crean a partir de clases.

💡 Analogía: Una clase es como un plano de construcción, y un objeto es la casa construida a partir de ese plano.

Sintaxis básica: La palabra clave class

Para definir una clase en Python usamos class:

class Player:
    """Una clase que representa un jugador en un videojuego."""

    def __init__(self, name, health=100):
        """Constructor: se ejecuta cuando creamos un objeto."""
        self.name = name
        self.health = health

    def take_damage(self, damage):
        """Método: reduce la salud del jugador."""
        self.health -= damage
        print(f"{self.name} recibió {damage} de daño. Salud: {self.health}")

    def heal(self, amount):
        """Método: restaura la salud del jugador."""
        self.health = min(self.health + amount, 100)
        print(f"{self.name} fue curado. Salud: {self.health}")

    def __str__(self):
        """Método especial: define cómo se muestra el objeto."""
        return f"{self.name} (Salud: {self.health})"

# Crear instancias (objetos)
player1 = Player("Aragorn", 100)
player2 = Player("Gimli", 95)

print(player1)  # Aragorn (Salud: 100)
player1.take_damage(30)  # Aragorn recibió 30 de daño. Salud: 70
player1.heal(20)  # Aragorn fue curado. Salud: 90

Conceptos clave

self — La referencia a la instancia

self representa el objeto actual. Todos los métodos reciben self como primer parámetro para acceder a los atributos del objeto.

__init__ — El constructor

Es el método especial que se ejecuta automáticamente al crear una instancia. Inicializa los atributos.

Variables de instancia vs. variables de clase

Las variables de instancia son únicas para cada objeto; las variables de clase se comparten entre todas las instancias:

class Product:
    """Producto de e-commerce."""
    tax_rate = 0.21  # Variable de clase: se comparte

    def __init__(self, name, price):
        self.name = name  # Variable de instancia: única por objeto
        self.price = price

product1 = Product("Arroz Bomba", 3.50)
product2 = Product("Arroz Integral", 4.20)

print(product1.name)  # Arroz Bomba
print(product1.tax_rate)  # 0.21 (accede a la variable de clase)

# Cambiar variable de clase (afecta a todas las instancias)
Product.tax_rate = 0.23

__str__ — Representación del objeto

Este método especial define cómo se muestra el objeto cuando lo imprimes o lo conviertes a string.

Herencia — Extender clases

Permite crear nuevas clases basadas en clases existentes, reutilizando código:

class Character:
    """Clase base."""
    def __init__(self, name, health):
        self.name = name
        self.health = health

    def introduce(self):
        return f"Soy {self.name}"

class Warrior(Character):
    """Clase derivada que hereda de Character."""
    def __init__(self, name, health, weapon):
        super().__init__(name, health)  # Llama al constructor de la clase padre
        self.weapon = weapon

    def attack(self):
        return f"{self.name} ataca con {self.weapon}!"

class Mage(Character):
    """Otra clase derivada."""
    def __init__(self, name, health, mana):
        super().__init__(name, health)
        self.mana = mana

    def cast_spell(self):
        if self.mana >= 20:
            self.mana -= 20
            return f"{self.name} lanza un hechizo!"
        return f"{self.name} no tiene maná suficiente."

warrior = Warrior("Conan", 150, "espada")
mage = Mage("Merlin", 80, 100)

print(warrior.introduce())  # Soy Conan
print(warrior.attack())  # Conan ataca con espada!
print(mage.cast_spell())  # Merlin lanza un hechizo!

super() — Acceder a la clase padre

super() permite llamar a métodos de la clase padre desde una clase derivada.

@property — Propiedades (getters)

El decorador @property permite acceder a métodos como si fueran atributos:

class Temperature:
    def __init__(self, celsius):
        self._celsius = celsius

    @property
    def fahrenheit(self):
        """Calcula Fahrenheit a partir de Celsius."""
        return (self._celsius * 9/5) + 32

    @property
    def celsius(self):
        return self._celsius

    @celsius.setter
    def celsius(self, value):
        self._celsius = value

temp = Temperature(25)
print(temp.celsius)  # 25
print(temp.fahrenheit)  # 77.0
temp.celsius = 30
print(temp.fahrenheit)  # 86.0

💻 Práctica Guiada

Paso 1: Crear la clase base

Vamos a construir una clase Product para un e-commerce de arroces:

class Product:
    """Representa un producto en la tienda."""

    def __init__(self, name, price, stock):
        self.name = name
        self.price = price
        self.stock = stock

    def __str__(self):
        return f"{self.name} - {self.price}€ ({self.stock} ud.)"

    def buy(self, quantity):
        """Compra una cantidad del producto."""
        if quantity <= self.stock:
            self.stock -= quantity
            return f"Compra realizada: {quantity}x {self.name}"
        return f"No hay suficiente stock. Disponibles: {self.stock}"

    def restock(self, quantity):
        """Aumenta el stock."""
        self.stock += quantity
        return f"Stock actualizado a {self.stock} ud."

# Probar
rice = Product("Arroz Bomba", 3.50, 50)
print(rice)  # Arroz Bomba - 3.5€ (50 ud.)
print(rice.buy(10))  # Compra realizada: 10x Arroz Bomba
print(rice)  # Arroz Bomba - 3.5€ (40 ud.)

Paso 2: Extender con herencia

Creamos una clase especializada PremiumProduct:

class PremiumProduct(Product):
    """Producto premium con descuento por volumen."""

    def __init__(self, name, price, stock, discount_threshold=5):
        super().__init__(name, price, stock)
        self.discount_threshold = discount_threshold

    def buy(self, quantity):
        """Sobrescribe el método de compra con descuento."""
        if quantity >= self.discount_threshold:
            discount = 0.15  # 15% de descuento
            final_price = self.price * (1 - discount)
            print(f"¡Descuento aplicado! Precio: {final_price}€/ud")

        return super().buy(quantity)

# Probar
premium = PremiumProduct("Arroz Integral Premium", 5.00, 30, discount_threshold=3)
print(premium.buy(5))  # ¡Descuento aplicado! Precio: 4.25€/ud
# Compra realizada: 5x Arroz Integral Premium

Paso 3: Usar @property

Agregamos una propiedad calculada:

class Product:
    # ... (código anterior)

    @property
    def total_value(self):
        """Calcula el valor total del inventario."""
        return self.price * self.stock

    @property
    def status(self):
        """Estado del stock."""
        if self.stock == 0:
            return "Agotado"
        elif self.stock < 10:
            return "Bajo stock"
        else:
            return "Disponible"

rice = Product("Arroz Bomba", 3.50, 15)
print(rice.total_value)  # 52.5
print(rice.status)  # Disponible

🧪 Mini-reto

Sistema simple de carrito de compra

Objetivo: Crea dos clases: Product y ShoppingCart con el siguiente comportamiento:

class Product:
    """Un producto con nombre, precio y stock."""

    def __init__(self, name, price):
        self.name = name
        self.price = price

    def __str__(self):
        return f"{self.name} ({self.price}€)"


class ShoppingCart:
    """Un carrito que almacena productos con cantidad."""

    def __init__(self):
        self.items = []  # Lista de tuplas (Product, quantity)

    def add_product(self, product, quantity=1):
        """Añade un producto al carrito."""
        self.items.append((product, quantity))
        print(f"Añadido: {quantity}x {product.name}")

    def total_price(self):
        """Calcula el precio total del carrito."""
        total = sum(product.price * quantity for product, quantity in self.items)
        return total

    def item_count(self):
        """Cuenta el número total de artículos."""
        return sum(quantity for product, quantity in self.items)

    def show_cart(self):
        """Muestra el contenido del carrito."""
        print("=== Carrito ===")
        for product, quantity in self.items:
            print(f"  {quantity}x {product}")
        print(f"Total: {self.total_price()}€")

# Prueba
rice1 = Product("Arroz Bomba", 3.50)
rice2 = Product("Arroz Integral", 4.20)

cart = ShoppingCart()
cart.add_product(rice1, 2)
cart.add_product(rice2, 1)
cart.show_cart()
print(f"Artículos: {cart.item_count()}")
💡 Pista: Usa una lista para almacenar tuplas (producto, cantidad).

Desafío bonus: Añade un método apply_discount(percentage) que reduce el total del carrito.
