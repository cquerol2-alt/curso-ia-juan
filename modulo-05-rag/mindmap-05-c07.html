Mind Map — Clase 07

Frontend con Streamlit para tu App RAG

Crea UIs para IA sin escribir HTML/CSS, todo en Python

❓
¿Por qué Streamlit?
+
Framework Python para UIs — escribe el frontend sin tocar HTML, CSS ni JavaScript
Perfecto para prototipos de IA — construir y iterar rápidamente
Se ejecuta con streamlit run app.py — un comando y tienes servidor local
Ideal para portfolios y demos — se ve profesional sin esfuerzo
🧩
Componentes básicos
+
st.title() — título grande para la página
st.text_input() y st.button() — inputs y botones
st.write() y st.markdown() — mostrar texto con formato
st.sidebar — panel lateral para controles secundarios
💬
Chat UI
+
st.chat_message("user"/"assistant") — burbujas de chat estilizadas
st.chat_input() — campo de texto para el usuario en la parte inferior
session_state — guardar el historial de mensajes entre ejecuciones
Loop que renderiza mensajes anteriores y captura nuevos en cada click
🔗
Integración con RAG
+
st.file_uploader() — widget para que el usuario suba PDFs
Procesar documentos en background con st.spinner()
Llamar a la chain de LangChain cuando se pregunta algo
Mostrar respuesta + fuentes con st.expander() para expandir/contraer
🎨
Personalización
+
st.set_page_config() — título e icono de la página
st.columns() — layouts en columnas para organizar contenido
st.slider() y st.selectbox() — controles de usuario interactivos
CSS custom con st.markdown(unsafe_allow_html=True) — controlar estilos avanzados
