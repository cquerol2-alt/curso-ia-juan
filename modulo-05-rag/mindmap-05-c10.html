Mind Map — Clase 10

Boss Battle: App RAG Completa

Integra todo lo aprendido en un proyecto real

🎯
El proyecto
+
"Pregúntale a Mis Apuntes": app que responde preguntas sobre tus PDFs
Portfolio project #2 — demuestra dominio de RAG end-to-end
Deployed y accesible por URL pública — funciona de verdad
🏗️
Arquitectura
+
Frontend: Streamlit con chat UI bonito
Backend: LangChain + ChromaDB para el pipeline RAG
LLM: OpenAI GPT-3.5/4 o Claude (con API keys)
Deploy: Render o Streamlit Community Cloud
✅
Funcionalidades MVP
+
Subir PDFs (mínimo 3 documentos de prueba)
Procesar y indexar automáticamente en ChromaDB
Chat con respuestas basadas en documentos indexados
Mostrar fuentes y números de página de cada respuesta
⭐
Funcionalidades Bonus
+
Panel de ajustes (cambiar modelo, temperatura, nº de fuentes)
Soporte multi-formato (PDFs, TXT, CSV, URLs)
Persistencia del vector store entre sesiones
Historial de conversaciones guardado
📋
Checklist de entrega
+
Código en GitHub con README profesional (instrucciones de setup)
App desplegada con URL funcional y accesible
Mínimo 3 documentos de prueba indexados y listos
Demostración grabada en vídeo (opcional pero recomendado para portfolio)
