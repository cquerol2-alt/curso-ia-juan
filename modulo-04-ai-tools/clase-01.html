Clase 01: GitHub Copilot — Setup y Autocompletado con IA

En este vídeo, Vida MRR te enseña todo lo que necesitas saber sobre GitHub Copilot: qué es, cómo instalarlo, cómo usar las sugerencias inline, Copilot Chat, y modos avanzados para potenciar tu productividad en VS Code.

📖 Resumen de conceptos

a) ¿Qué es GitHub Copilot?

GitHub Copilot = Tu asistente de programación con IA. Es un servicio de GitHub y OpenAI que usa modelos de IA para sugerirte código mientras escribes. Piensa en él como un "autocomplete sobreesteroides".

  • Integrado en VS Code: Funciona directamente en tu editor
  • Basado en GPT: Entiende contexto, nombres de variables, y patrones
  • Plan gratuito disponible: 2,000 autocompletados/mes + 50 mensajes de chat/mes (limitado pero útil para empezar)
  • Plan pro: Acceso ilimitado, $10/mes o gratis si eres estudiante universitario
💡 Consejo: Como estudiante, puedes obtener GitHub Copilot gratis si verificas tu email universitario en GitHub Student Developer Pack.

b) Setup paso a paso en VS Code

Paso 1: Tener una cuenta de GitHub

  1. Ve a github.com/signup
  2. Crea tu cuenta si aún no tienes (es gratis)
  3. Verifica tu email

Paso 2: Instalar la extensión en VS Code

  1. Abre VS Code
  2. Ve a "Extensions" (icono de bloques a la izquierda o Ctrl+Shift+X)
  3. Busca "GitHub Copilot"
  4. Haz clic en "Install" (la versión oficial, con el logo de GitHub)
  5. Espera a que termine la instalación

Paso 3: Iniciar sesión con GitHub

  1. VS Code te pedirá que inicies sesión. Haz clic en "Sign in with GitHub"
  2. Se abrirá una pestaña en el navegador
  3. Autoriza la extensión para acceder a tu cuenta
  4. Volverá a VS Code y verás un ✓ verde indicando que está conectado

Paso 4: Activar el plan gratuito (o Student)

  1. Ve a github.com/copilot
  2. Haz clic en "Get access to Copilot" o "Start free trial"
  3. Elige "Free" (limitado) o sigue el flujo de Estudiante si tienes email universitario
  4. Confirma tu plan
  5. ¡Listo! VS Code ya debería mostrar sugerencias de Copilot
⚠️ Nota importante: Si el plan de prueba expira o usas los 2,000 autocompletados/mes, Copilot seguirá funcionando pero sin sugerencias hasta que renueves. No hay sorpresas de facturación.

c) Tipos de sugerencias: Inline, Chat, y Slash Commands

1. Sugerencias Inline (el uso más común)

Mientras escribes código, Copilot te sugiere automáticamente. Ejemplo:

// Escribes un comentario descriptivo:
// función que valida un email

// Copilot sugiere automáticamente:
function validarEmail(email) {
    return /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email);
}

Controles:

  • Tab: Aceptar la sugerencia
  • Esc: Rechazar la sugerencia
  • Alt + [/]: Navegar entre sugerencias alternativas (si hay varias)
  • Ctrl+Alt+[/]: Ver sugerencias alternativas en el editor

2. Copilot Chat (preguntas dentro de VS Code)

Abre el panel de chat para hacer preguntas o pedir ayuda sin dejar el editor.

  • Ctrl+I: Abrir Copilot Chat inline (pregunta rápida)
  • Ctrl+L: Limpiar el chat
  • O usa el panel lateral: icono de chat en la izquierda (debajo de Extensions)

Ejemplo de uso:

// Abre Chat (Ctrl+I) y pregunta:
"Explícame qué hace esta función"
"Refactoriza este código para hacerlo más eficiente"
"Escribe tests para esta función"

3. Slash Commands (órdenes especiales en Chat)

  • /explain: Explica el código seleccionado
  • /fix: Sugiere correcciones al código
  • /tests: Genera tests automáticos
  • /doc: Añade documentación
  • /optimize: Mejora el rendimiento

Ejemplo:

// Selecciona el código, abre Chat y escribe:
/explain
// Copilot te explicará línea por línea

d) Modos de Copilot (2025+): Ask, Edit, Agent

GitHub Copilot ha evolucionado con varios modos de interacción:

  • Ask Mode: Haces preguntas sobre el código. Copilot responde con explicaciones o sugerencias
  • Edit Mode: Seleccionas código y pides cambios directos ("hazlo 30% más rápido", "añade validación")
  • Agent Mode: Copilot puede trabajar en múltiples archivos, entender el contexto del proyecto, hacer cambios complejos

Los modos se activan según el contexto. En desarrollo temprano usarás Ask y Edit principalmente.

e) @workspace y participantes: Contexto avanzado

En Copilot Chat, puedes usar @mentions para darle más contexto:

  • @workspace: Copilot analiza tu proyecto entero (archivos, estructura, dependencias)
  • @terminal: Proporciona contexto de lo que ejecutaste en la terminal
  • @vscode: Información sobre la configuración de VS Code

Ejemplo:

// En Copilot Chat, escribe:
"@workspace, cómo puedo mejorar la estructura de carpetas de este proyecto?"

// Copilot analiza todo y da recomendaciones contextualizadas

f) Ejemplo práctico: Generar una función con Copilot

Paso 1: Abre un archivo nuevo (main.py o main.js)

Paso 2: Escribe un comentario descriptivo:

// función que ordena un array de números de mayor a menor

Paso 3: Presiona Enter y espera a que Copilot sugiera

Resultado esperado:

function ordenarDescendente(arr) {
    return arr.sort((a, b) => b - a);
}

Paso 4: Presiona Tab para aceptar o Esc para rechazar

Eso es todo. Copilot entiende la intención del comentario y genera código válido. ¡Así es la programación 2025!

💡 Consejo de productividad: Cuanto más específico es el comentario, mejor es la sugerencia. "función que valida un email con regex y retorna true/false" genera código mejor que solo "función".

💻 Práctica guiada

Ejercicio 1: Instalar y configurar GitHub Copilot (15 min)

Objetivo: Tener Copilot funcionando en VS Code con acceso activo.

  1. Crear/verificar cuenta GitHub
    • Si no tienes cuenta, ve a github.com/signup
    • Crea tu cuenta y verifica el email
  2. Instalar extensión en VS Code
    • Abre VS Code
    • Presiona Ctrl+Shift+X (Extensions)
    • Busca "GitHub Copilot" (oficial, con logo de GitHub)
    • Haz clic en "Install"
  3. Autorizar con GitHub
    • VS Code abrirá un diálogo de login
    • Haz clic en "Sign in with GitHub"
    • Completa el flujo en el navegador
    • Autoriza la extensión
    • Vuelve a VS Code (debería ver el ✓ verde)
  4. Activar plan gratuito
    • Ve a github.com/copilot
    • Haz clic en "Get access to Copilot" o "Start free trial"
    • Elige "Free plan" (2,000 completados/mes)
    • O si eres estudiante, verifica tu email universitario en GitHub Student Pack para acceso ilimitado gratis
    • Confirma y ¡listo!
  5. Verificación: Abre VS Code y escribe un comentario. Copilot debería sugerir código. Si ves las sugerencias grisáceas, ¡funciona!

Ejercicio 2: Usar sugerencias inline (20 min)

Objetivo: Escribir un proyecto pequeño usando SOLO comentarios, dejando que Copilot genere el código.

Proyecto: Calculadora Simple

Crea un archivo calculadora.py (o .js):

# Proyecto: Calculadora con funciones básicas
# Descripción: suma, resta, multiplicación, división

# función que suma dos números y retorna el resultado
def suma(a, b):
    return a + b

# función que resta dos números
def resta(a, b):
    return a - b

# función que multiplica dos números
def multiplica(a, b):
    return a * b

# función que divide dos números (con manejo de división por cero)
def divide(a, b):
    if b == 0:
        return "Error: no se puede dividir entre cero"
    return a / b

# función principal que pide números al usuario y muestra opciones
def main():
    print("=== Calculadora Simple ===")
    print("1. Suma")
    print("2. Resta")
    print("3. Multiplicación")
    print("4. División")

    opcion = input("Elige una opción (1-4): ")
    num1 = float(input("Primer número: "))
    num2 = float(input("Segundo número: "))

    if opcion == "1":
        print(f"Resultado: {suma(num1, num2)}")
    elif opcion == "2":
        print(f"Resultado: {resta(num1, num2)}")
    elif opcion == "3":
        print(f"Resultado: {multiplica(num1, num2)}")
    elif opcion == "4":
        print(f"Resultado: {divide(num1, num2)}")
    else:
        print("Opción inválida")

# llamar a la función principal
if __name__ == "__main__":
    main()

¿Cómo hacerlo con Copilot?

  1. Escribe solo los comentarios descriptivos (las líneas con #)
  2. Presiona Enter después de cada comentario
  3. Copilot debería sugerir el código (sugerencias grisáceas debajo)
  4. Presiona Tab para aceptar, Esc para rechazar
  5. Si no te gusta la sugerencia, continúa escribiendo tu código y pide otra

Tiempo esperado: 15-20 minutos si dejas que Copilot genere la mayoría del código.

Ejercicio 3: Usar Copilot Chat (15 min)

Objetivo: Usar /slash commands para mejorar y entender el código.

  1. Explicar código con /explain
    • Selecciona la función divide() del ejercicio anterior
    • Abre Copilot Chat: Ctrl+I (inline) o panel lateral
    • Escribe: /explain
    • Copilot te explicará línea por línea qué hace
  2. Generar tests con /tests
    • Con la función suma() seleccionada
    • Abre Chat y escribe: /tests
    • Copilot generará un archivo de tests (pytest si es Python, jest si es JavaScript)
    • Copia el código generado a un archivo test_calculadora.py
  3. Refactorizar con /fix
    • Selecciona la función main()
    • En Chat, escribe: /fix mejora el manejo de errores
    • Copilot te sugerirá mejoras y código refactorizado
  4. Bonus: Usar @workspace
    • En Chat escribe: @workspace, ¿cómo puedo estructurar mejor este proyecto de calculadora?
    • Copilot analizará tu código y sugerirá mejoras de arquitectura
⏱️ Tiempo total sugerido: 50 minutos

Objetivo: Crear un archivo con 10 funciones útiles escribiendo SOLO comentarios descriptivos, dejando que Copilot genere el 100% del código.

  1. Validar si una cadena es un email válido
  2. Formatear una fecha (ej: "01/04/2026" → "1 de abril de 2026")
  3. Ordenar un array de objetos por propiedad
  4. Convertir camelCase a snake_case
  5. Contar palabras en un texto
  6. Generar un ID único (UUID simple)
  7. Validar si una contraseña es fuerte (mín 8 caracteres, mayúscula, número)
  8. Paginar un array (ej: página 2 de 10 elementos por página)

  4. Tab para aceptar o rechazar
  5. Si la sugerencia no es perfecta, ajusta manualmente o pide una nueva
