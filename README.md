# Curso IA para Desarrolladores Web

Curso a medida para Juan Querol León

---

## Descripción

Curso de 10 semanas (100 horas) para transformar a un desarrollador web en un **desarrollador potenciado por IA**.

Clases interactivas en HTML con vídeo, prácticas, quizzes y gamificación. Aprende a integrar inteligencia artificial en tu flujo de desarrollo, desde prompts hasta agentes autónomos.

---

## Stack Tecnológico

- HTML5
- CSS3
- JavaScript vanilla (sin frameworks)

---

## Características

- **Quizzes interactivos** — 7 de 10 preguntas para pasar
- **Sistema XP** — Acumula experiencia y sube de nivel
- **Badges** — Desbloquea insignias por hitos
- **Streak tracking** — Racha de aprendizaje diario
- **Boss Battles** — Reto final de cada módulo
