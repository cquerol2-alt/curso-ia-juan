📚 Resumen de Conceptos

Tu Portfolio Web es tu Escaparate Profesional

La primera impresión es fundamental. Un portfolio web bien hecho puede ser la diferencia entre ser ignorado o entrevistado. Es donde muestras tu work, tu forma de pensar y tu capacidad de comunicación.

Opciones para Crear tu Portfolio

1. GitHub Pages + HTML/CSS/JS
  • ✓ Gratis (límite de dominio .github.io)
  • ✓ Simple de desplegar
  • ✓ Rápido
  • ✓ Control total del código
  • ⭐ Recomendado para empezar
2. Next.js + Vercel
  • ✓ Más profesional
  • ✓ Rendimiento optimizado
  • ✓ Lo que ya aprendiste en Módulo 7
  • ⚠ Requiere más setup
3. Templates Listos
  • Hugo, Astro, o templates GitHub
  • ✓ Rápido si no quieres customizar
  • ⚠ Menos control del código

Secciones Esenciales de tu Portfolio

  • Hero: Tu nombre, título, foto/avatar, tagline impactante
  • Sobre Mí: 2-3 párrafos. Tu historia y motivación.
  • Proyectos: Cards con screenshot, descripción, tech stack, demo + GitHub
  • Skills: Visualización clara de tu stack técnico (no solo lista aburrida)
  • Certificaciones: Badges/logos de cursos, certificados, comunidades
  • Contacto: Email, LinkedIn, GitHub, link al CV

El Vídeo de Presentación de 2 Minutos

¿Por qué? Humaniza tu perfil, muestra comunicación, te diferencia del 99%.

Estructura recomendada:

  1. 0:00-0:20 — Quién eres y qué haces
    Ejemplo: "Hola, soy Juan, desarrollador full-stack con pasión por la IA"
  2. 0:20-0:50 — Tu stack técnico y tipos de proyectos
    Ejemplo: "Trabajo con Python, React, Node.js y LLMs"
  3. 0:50-1:30 — Demo rápida de tu mejor proyecto
    Screen recording de tu portfolio o app en vivo (sin audio, subtítulos)
  4. 1:30-2:00 — Qué buscas y cómo contactarte
    Ejemplo: "Busco proyectos que combinen desarrollo y AI. LinkedIn: [link]"

Tips técnicos:

  • Buena iluminación (luz frontal, sin sombras)
  • Fondo limpio (pared blanca o blur de escritorio)
  • Micrófono decente (los AirPods sirven, no uses el del laptop)
  • Editar con CapCut (gratis, fácil, exporta HD)
  • Subir a YouTube (unlisted si no quieres que sea público)

⚔️ Misión: Construye tu Portfolio Profesional

En este Boss Battle, vas a crear tu portfolio web completo Y grabar el vídeo de presentación de 2 minutos. Este es tu escaparate ante potenciales empleadores y clientes.

⚙️ Pasos a Seguir

  1. 1. Crear repo GitHub: Tu nombre de usuario seguido de .github.io (ej: juanquerol.github.io)
  2. 2. Estructura HTML: Hero, About, Projects (mínimo 3), Skills, Certifications, Contact
  3. 3. Estilos CSS: Dark theme, responsive (mobile-first), animaciones suaves
  4. 4. Proyectos: Cada proyecto necesita screenshot, descripción, tech stack y links
  5. 5. Deploy: Subir a GitHub Pages y verificar que sea accesible por URL pública
  6. 6. Grabar vídeo: 2 minutos presentándote, tu stack y demostrando tu mejor proyecto
  7. 7. Subir a YouTube: Unlisted, embeber en tu portfolio y compartir en LinkedIn Featured
